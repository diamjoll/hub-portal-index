# hub-portal-index

本仓库用于归档和发布多个独立 HTML 页面，作为一组通用前端资源的集中索引入口。

## 项目简介

`hub-portal-index` 是一个轻量级的页面集合仓库。它不绑定任何特定域名或服务，仅用于存放结构清晰、可独立访问的 HTML 文件。这些页面可以是工具界面、信息展示页、导航页或任何形式的静态前端内容。

仓库的主要目的是提供一个整洁的索引环境，方便维护与分发多个零散的 HTML 页面。

## 目录说明

仓库根目录下包含以下内容：

- `index.html` — 主入口页，通常作为页面列表或导航
- `pages/` — 存放各个独立 HTML 页面的文件夹
- `assets/` — 公共静态资源（CSS、JavaScript、图片等）
- `README.md` — 本文件

每个 HTML 页面应尽量自包含，若依赖外部资源，建议引用 `assets/` 目录下的文件。

## 页面归档说明

- 所有页面均为独立 HTML 文件，可直接在浏览器中打开使用。
- 页面之间无强依赖关系，可单独更新或删除。
- 归档的页面应保持基础的可读性与功能性，不包含恶意或违反平台政策的内容。
- 页面命名建议使用小写英文，用连字符分隔，例如 `example-tool.html`。

## 维护说明

- 欢迎提交 Pull Request 添加新的 HTML 页面或改进现有页面。
- 请确保新增页面不依赖外部不可控资源（如第三方 CDN），或已在 `assets/` 中提供备份。
- 不建议在页面中加入跟踪、统计或广告代码，以保持仓库的通用性。
- 若需修改目录结构或资源引用方式，请先开 Issue 讨论。
